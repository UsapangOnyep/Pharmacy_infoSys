Pharmact_POS.bat use for opening POS in Incognito Mode.
Update the path of app server in Pharmacy_POS.bat.

To create .exe (executable file), download and install Bat to Exe Converter (https://www.majorgeeks.com/files/details/bat_to_exe_converter.html)
-> click open
-> select Pharmacy_POS.bat
-> Run to Test | Convert to Create .exe

To convert png to ico, use PNG2ICO.
